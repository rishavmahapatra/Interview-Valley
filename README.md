# Interview Valley - Frontend

Interview Valley is a modern **mock interview platform frontend** designed to help users practice and prepare for technical interviews through an intuitive and engaging user experience.

## 🚀 Overview

This repository contains the frontend application for **Interview Valley**, a platform that simulates real interview experiences to help candidates improve their interview skills, build confidence, and track their progress.

## ✨ Features

* 🔐 User Authentication and Authorization
* 🎯 Interactive Mock Interview Sessions
* 📊 Performance Feedback and Insights Dashboard
* 📱 Responsive Design for Desktop and Mobile Devices
* 🎨 Modern and Clean User Interface
* ⚡ Fast and Optimized User Experience
* 👤 User Profile Management
* 📈 Interview History Tracking

## 🛠️ Tech Stack

* **Framework:** React.js
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **Routing:** React Router
* **State Management:** Context API
* **Authentication:** JWT

## ⚙️ Getting Started

### Prerequisites

Make sure you have the following installed:

* Node.js (v18 or later)
* npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/rishavmahapatra/Interview-Valley.git
```

2. Navigate to the project directory:

```bash
cd Interview-Valley
```

3. Install dependencies:

```bash
npm install
```

or

```bash
yarn install
```

4. Start the development server:

```bash
npm run dev
```

or

```bash
yarn dev
```

The application will be available at:

```text
http://localhost:5173
```

## 🏗️ Build for Production

To generate a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
